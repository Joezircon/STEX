# STEX Roadmap

## v0.1 - Foundation

- Desktop window
- Open PNG/JPG/BMP
- Zoom and pan
- Topology check
- White island detection
- Save PNG

## v0.2 - Rib Sketch

- Draw rib preview
- Accept or reject rib
- Adjustable rib width
- Recheck topology automatically

## v0.3 - Paint Selection

- Paint feature selections
- Save named features
- Add/subtract selection brush

## v0.4 - Feature Tree

- Named feature hierarchy
- Isolate selected feature
- Topology check per feature

## v0.5 - Export

- SVG export
- Cricut-friendly output
- Layer export

## v1.0 - First Public Release

- Installer
- Documentation
- Example projects
- Stable workflow
