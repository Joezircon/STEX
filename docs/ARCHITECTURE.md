# STEX Architecture

## Major Modules

- Canvas Engine
- Topology Engine
- Rib Engine
- Selection Engine
- Feature Engine
- Export Engine
- Project Engine
- UI

## Core Philosophy

Each module should have one responsibility.

## Topology Engine

Finds unsupported white islands and structural warnings.

## Rib Engine

Creates, previews, edits, and validates ribs.

## Selection Engine

Lets the user define named features.

## Export Engine

Produces Cricut/laser/vinyl-friendly files.
