# STEX UI Style

## Core Aesthetic

STEX should feel like an old arcade game mixed with professional stencil CAD.

## Visual Direction

- 8-bit graphics
- Synthwave colors
- Graffiti energy
- Dark arcade cabinet background
- Neon highlights
- Pixel-style icons
- CRT-inspired glow
- Clean enough for serious work

## Interface Language

Use playful arcade-inspired labels where appropriate:

- INSERT IMAGE
- TOPOLOGY CHECK
- RIB SKETCH
- ZERO ISLANDS
- CUT READY
- EXPORT LEVEL
- STEX VERIFIED

## UI Rule

The interface can be fun, but the tools must remain clear and professional.

## Win Condition

A finished stencil should feel like completing a level:

ZERO WHITE ISLANDS.

STEX VERIFIED.
