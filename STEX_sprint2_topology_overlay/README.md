# STEX Sprint 2 - Topology Overlay

This update connects the Forge Engine to the STEX canvas.

## New Features

- TOPOLOGY CHECK now analyzes the loaded image.
- Unsupported white islands are detected.
- Islands are drawn as red overlays.
- Islands are numbered.
- Inspector log reports island count.
- Status bar shows CUT READY when no critical islands are found.

## Run

Double-click:

```text
run_STEX.bat
```

or:

```bash
python -m src.stex.main
```
