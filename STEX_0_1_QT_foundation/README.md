# STEX 0.1 Qt Foundation

This is the first PySide6/Qt foundation build for STEX.

## Current Features

- Launches a real desktop window
- Arcade/synthwave-inspired dark UI
- Left tool panel
- Main canvas placeholder
- Status bar
- INSERT IMAGE button placeholder
- TOPOLOGY CHECK button placeholder
- RIB SKETCH button placeholder

## Run

Double-click:

```text
run_STEX.bat
```

or run:

```bash
python -m src.stex.main
```
