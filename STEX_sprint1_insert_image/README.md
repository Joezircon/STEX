# STEX Sprint 1 - Insert Image

This update adds the first real canvas capability.

## New Features

- INSERT IMAGE opens a file picker.
- PNG/JPG/BMP images load onto the canvas.
- Mouse wheel zooms.
- Right mouse drag pans.
- Reset View button centers the image.
- Status bar shows image size and zoom percent.

## Run

Double-click:

```text
run_STEX.bat
```

or:

```bash
python -m src.stex.main
```
